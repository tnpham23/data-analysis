House Sales and Leukemia

Stata: Univariate Linear Regression

In this APA style Economics paper, I explore the data between two counties, Churchhill (treatment variable with existing leukemia cases) and Lyon (control variable with no leukemia cases). With the variable of interest being the log real house sales in these communities, we are curious if there is a linear relationship between house sales and quality of life, in this case health.

STATA Code for Figures

tabstat lnrealsales if county==1, by(y00) stat(mean n sd min max)
tabstat lnrealsales if county==0, by(y00) stat(mean n sd min max)

tabstat age acres sqft condition cases if county==1, by(y00)
tabstat age acres sqft condition cases if county==0, by(y00)

**table5
reg lnrealsales county age cases acres sqft condition y1 y2 y3 y4 y5 y6 y7 y8 y9 y10 y11 y12
test ( y1 y2 y3 y4 y5 y6 y7 y8 y9 y10 y11 y12)

**table6
reg lnrealsales county y00 y00county

**table7
reg lnrealsales county y00 y00county age cases acres sqft condition


graph twoway kdensity lnrealsales if y00 || kdensity lnrealsales if !y00, ///
		title("Distribution of Log Real House Prices") ///
		xtitle("Log Real Sales") ///
		ytitle("Density") ///
		legend(label(1 "Year >=2000") label(2 "Year < 2000"))

reg lnrealsales county cases y00 y00county y00cases
est store m1

reg lnrealsales county cases y00 y00county y00cases, robust
est store m2

reg lnrealsales county cases y00 y00county y00cases sqft age acres condition
est store m3

esttab m1 m2 m3 using "mreg.rtf", se label ar2 replace ///
		title("Explaining Change of House Prices between 1990-2002") ///
		mtitles("OLS" "Robust Std" "With Controls")
